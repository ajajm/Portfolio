# Credentials Folder

This folder contains credential images and PDFs for the portfolio.

## Instructions:

1. Add your credential images here with the following names:
   - `osc-contributor.jpg` - Open Source Connect Contributor Certificate
   - `osc-state-lead.jpg` - Open Source Connect State Lead Certificate
   - `osc-campus-lead.jpg` - Open Source Connect Campus Lead Certificate
   - `google-ambassador.jpg` - Google Student Ambassador Certificate

2. You can use JPG, PNG, or PDF formats

3. Update the file extensions in `src/pages/Credentials.jsx` if needed

4. For Credly badges, update the URL in the Credentials component to point to your actual Credly profile
